💄 Sistematización Beauty Company

Este proyecto busca **sistematizar los procesos de la empresa Beauty Company**, ofreciendo herramientas digitales que faciliten la gestión de clientes, productos y servicios.

---

## 🚀 Tecnologías utilizadas
- ⚙️ Java / JavaFX → interfaz gráfica y lógica del sistema  
- 🗄️ MySQL / MongoDB → gestión de base de datos  
- 🔐 OWASP Guidelines → seguridad y validaciones  
- 🔄 Git + GitHub → control de versiones y colaboración
